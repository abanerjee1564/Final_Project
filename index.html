The Human Experience

Hello world! My name is Akash Banerjee - I'm a graduating senior economics major at Temple University seeking to better understand the human experience. I originally created this website as part of an assignment for my Web Mapping and GIS class. In the class, I learned how to create user-friendly, interactive maps and charts using web-based tools such as JavaScript, Leaflet, and ArcGIS SDK. In this class, I have made several achievements I had never thought myself capable of, from designing my own website (yes, this one) to creating informative, interactive maps from scratch (i.e. just from writing code). On a higher level, this semester I juggled a full courseload while interning part-time at the Federal Reserve Bank of Philadelphia and working on a national case competition called the Fiscal Challenge, where we were tasked with stabilizing the federal debt to GDP ratio to 95% by 2054 (and we won!). I'm proud of the work I've put in these past four years, but I know this is only the beginning of my personal and professional journey. That's why I'm devoting this simple blog to something bigger - an exploration of the human experience, where I share data, stories, and insights I've gathered from my small corner of the world.

Surveys

We've all been asked to take a survey at some point. While the concept of a survey (a tool to gauge human sentiment and behavior) is fundamentally sound, many surveys are superficial and/or biased (e.g. many political opinion surveys). This has inspired me to come up with some of my own surveys, which I'll be posting on my very fancy Google Site (link below). The goal of these surveys is to learn more about the human experience - the internal and external forces that drive us towards (or away from) feeling at peace in a chaotic world. Topics will include (but are not limited to) preferences, desires, personal experience, and most importantly, perspectives on how to live the "Good Life."

Simple Map of Three Cities

This is a basic map showing the locations of three cities: Topeka, Kansas; Louiseville, Kentucky; and Houston, Texas. When you click on the marker, you can see the name of the city and state. This is the first map I created in GUS 4064 - I learned how to use Amazon Q and create a basic interactive map using Visual Studio code.

Simple Map of Two Cities with Population Data

This map depicts two cities, Lancaster and Harrisburg. The size of each tooltip reflects the corresponding city's population - clicking on the tooltip shows the user the city name and population. In this assignment, I learned how to customize tooltips based on a given criteria.

Parks and Recreation Sites in Philadelphia

This map shows Parks and Recreation sites in Philadelphia using geojson data from OpenData Philly. Each green circle represents a unique park or recreation site. In this assignment, I learned how to access and incorporate geojson data into interactive maps.

U.S. Power Plants

This is an interactive map demonstrating electric production capacity in MW for biomass, geothermal, and solar plants. I created this using data from the World Resources Institute Global Power Plant Database. In this assignment, I learned how to create a proportional symbol map and optimize interactive maps using vector instead of raster tiles.

Ratio of Public to Private Schools (by State)

This is a map showing the ratio of public to private schools by state using data from the ArcGIS REST Services Directory. Users can click on a state to see the total number of public and private schools. In this assignment, I learned how to use the ArcGIS SDK environment to create an interactive choropleth map.

Poverty Index by County

This map shows the county poverty index by U.S. county using data from the ArcGIS REST Services Directory. The size of the green circlemarkers reflect the magnitude of the poverty index, and clicking on a circlemarker allows the user to see the poverty index score and percent living below poverty in a given county. In this assignment, I learned how to make a proportional symbol map using the ArcGIS SDK environment.

Find the Hidden Jewel: Interactive Game

In this interactive game, you will try to find the Hidden Jewel by clicking on locations until the jewel's icon appears. As a hint, it is one of the Seven Wonders of the World! In this assignment, I learned how to create a "hidden" marker that appears when the user clicks on the correct location. This also involved learning how to condition pop-up content on certain criteria, such as distance from clicked location to correct location.

Exploration of Google Analytics

Here, I present two interactive charts I created using Google Charts tools made available through Javascript. The first chart plots human births and deaths from 2018 - 2021 using CDC data, while the other chart plots my hypothesized relationship between happiness and love. In this assignment, I learned how to use JS documentation to create interactive charts.
